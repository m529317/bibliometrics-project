# bibliometrics-project
# 文献计量学小组协作项目

## 项目背景
本仓库用于小组5人协作完成人工智能教育领域研究文献计量与知识图谱分析

## 研究方向
人工智能教育领域研究文献计量与知识图谱分析

## 整体流程
1. 检索文献 → 2. 筛选去重 → 3. 数据清洗 → 4. 数据分析 → 5. 可视化与论文撰写

## 使用教程
1. 克隆仓库：`git clone https://github.com/m529317/bibliometrics-project.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 各成员将成果放入对应文件夹，并提交Pull Request

## 协作规范
- 分支命名：`feature/任务名`、`fix/问题描述`
- Commit消息格式：`[模块] 简短描述`
- 重要成果请通过Pull Request合并到main分支
